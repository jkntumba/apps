With a token saved, Stacks fetches covers from Discogs first (most accurate for vinyl) and checks live listings for wanted records, popping a notification when it finds any. Without one, “Check listings” opens Discogs & eBay searches and covers still work via Discogs, just slower (rate-limited).

Pulls real album covers for every record that doesn't have one yet. With a Discogs token saved above it searches Discogs first (most accurate for vinyl); otherwise it uses the iTunes catalog — now artist-verified to avoid wrong matches — and falls back to Discogs. For a large library this can take a few minutes — keep the app open while it runs. If a single cover comes out wrong, open that record and tap “💿 From Discogs”. You can run it again any time to retry any that weren't found.
