Library tools

Your library lives on this device. Back it up or move it with a library file.

Without a token, “Check listings” opens Discogs & eBay searches. With one, Stacks checks live vinyl listings for wanted records and pops a notification when it finds any.

Pulls real album covers from the iTunes catalog for every record that doesn't have one yet. For a large library this can take a few minutes — keep the app open while it runs. You can run it again any time to retry any that weren't found.
